

# Project Title

Pupper Newsfeed



 Access my site at https://pupper-newsfeed-front.onrender.com

## About The App

[Pupper newsfeed] is website you can upload posts about your dogs by filling on a form with different categories. The user can log in using google authentication. The user can like posts, delete posts and edit posts.

## Approach

Build with react and styled with tailwind. Used react router to navigate pages and redux for state management. google auth was used for authentication. The server was built with Nodejs using the express framework and mongo as the database.
